# 快速解决方案 - "正在分析操作"卡住

## 🎯 立即解决（30秒）

### 方案1：等待30秒（推荐）

```
1. 侧边栏显示"正在分析操作..."
2. 等待 30 秒
3. 会自动显示默认描述选项
4. 继续使用 ✅
```

**为什么会这样？**
- 如果没有配置API，会在30秒后超时
- 超时后自动使用默认选项
- 这是在等待网络请求超时

### 方案2：刷新侧边栏（更快）

```
1. 关闭侧边栏（点击右侧的X）
2. 再次点击插件图标
3. 点击 "打开编辑器"
4. 应该直接显示描述选项
```

### 方案3：配置API（可选，更好效果）

```
1. 点击插件 → "设置"
2. 填写API配置：
   API Key: sk-xxxxx
   Base URL: https://api.deepseek.com
   模型名称: deepseek-chat
3. 保存
4. 重新录制
```
