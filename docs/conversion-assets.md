## 推荐首屏文案

> 录一次网页操作，自动生成带截图的操作文档。  
> SmartPages 是一个开源浏览器插件，适合 QA 写复现步骤、产品写操作手册、客服写知识库。

## README 转化顺序

1. 一句话价值说明。
2. 15 秒 GIF。
3. 三个典型场景：QA、产品/实施、客服。
4. 示例文档链接。
5. 安装和配置步骤。
6. GitHub issue / Star 引导。

## 可复制短介绍

SmartPages 是一个开源浏览器插件，可以录制真实网页操作流程，自动捕获点击、输入、页面跳转和截图，然后调用你配置的 AI 模型生成 Markdown、HTML 或纯文本操作文档。它适合 QA、产品、实施、客服和运营团队，用来快速生成 Bug 复现说明、后台操作手册、培训材料和知识库文章。
