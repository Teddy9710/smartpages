## 发帖时怎么用

- 掘金、cnblogs：把示例作为“生成结果”小节，说明不是概念 demo，而是实际可交付文档。
- V2EX、即刻：截取一段示例结果，配合 GIF 展示“录制一次 -> 自动生成”。
- 小红书：优先展示“前后对比”，例如“原来要手写 20 分钟，现在录一遍直接出初稿”。
