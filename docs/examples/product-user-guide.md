# 示例：创建优惠券操作手册

> 适用场景：产品、实施或运营人员录制一次后台操作流程，使用 SmartPages 生成可交付给团队或客户的操作手册。

## 操作目标

在营销后台创建一张满减优惠券，并设置领取时间、使用门槛和发放数量。

## 操作前准备

- 已登录营销后台。
- 当前账号拥有“优惠券管理”权限。
- 已确认活动名称、发放数量、有效期和优惠规则。

## 操作步骤

### 1. 进入优惠券管理

在左侧导航栏中点击“营销中心”，然后进入“优惠券管理”。  
[截图1]

### 2. 新建优惠券

点击页面右上角的“新建优惠券”按钮，进入优惠券创建页面。  
[截图2]

### 3. 填写基础信息

填写优惠券名称，例如“618 满 200 减 30”。选择优惠券类型为“满减券”。  
[截图3]

### 4. 配置优惠规则

在“使用门槛”中输入 `200`，在“优惠金额”中输入 `30`。  
[截图4]

### 5. 设置领取和使用时间

设置领取开始时间、领取结束时间，并选择优惠券可使用的日期范围。  
[截图5]

### 6. 设置发放数量

在“发放总量”中输入本次活动计划发放的优惠券数量，例如 `1000`。  
[截图6]

### 7. 保存并发布

检查所有配置无误后，点击“保存并发布”。系统提示发布成功后，优惠券会出现在列表中。  
[截图7]

## 验证方式

返回优惠券列表，搜索优惠券名称，确认状态为“已发布”，并检查面额、门槛、库存和有效期是否正确。

## 注意事项

- 发布后如已有用户领取，部分字段可能无法修改。
- 活动开始前建议先使用测试账号领取并下单验证一次。
- 如果优惠券涉及多个渠道，请确认渠道限制是否配置正确。
