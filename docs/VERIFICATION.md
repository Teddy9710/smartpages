## 使用说明

### 通过设置界面上传文档
1. 点击扩展图标
2. 选择"设置"选项
3. 在"文档管理"部分上传文件
4. 管理已上传的文档

### 通过命令行接口管理文档
在浏览器控制台中使用：
```javascript
scribe help                    // 查看帮助
scribe upload ./file.txt       // 上传文档
scribe list                    // 列出文档
scribe search "keyword"        // 搜索文档
scribe get <doc_id>           // 获取文档详情
scribe delete <doc_id>        // 删除文档
```

### 通过API接口管理文档
```javascript
const system = new DocumentManagementSystem();
await system.uploadDocument(file, options);
const docs = await system.searchDocuments('query');
```

- 文件类型验证防止恶意文件上传
- 适当的数据清理和过期机制
